# multipage-cli

## 开发环境
使用es
```
npm run dev
```
使用typescript
```
npm run dev:ts
```


## 打包构建
使用es
```
npm run build
```
使用typescript
```
npm run build:ts
```
    
## 开发目录结构
```
├─static
│  ├─css
│  │  ├─common
│  │  └─page
│  ├─img
│  ├─js
│  │  ├─page
│  │  └─util
│  └─ts
│      ├─page
│      └─typings
└─views
```
